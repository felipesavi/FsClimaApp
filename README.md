⛅ Aplicación FsClima
FsClima es una aplicación móvil desarrollada con React Native que permite consultar información meteorológica de manera rápida y sencilla. El proyecto está enfocado y optimizado para dispositivos Android.

🚀 Características
Consulta de clima: Visualización de condiciones climáticas actuales.

Interfaz intuitiva: Diseño sencillo con componentes modales y listas de datos.

Optimizado para Android: Configuración lista para ser ejecutada en emuladores o dispositivos físicos Android.

🛠️ Tecnologías
React Native (Core)

JavaScript (ES6+)

NPM (Gestor de paquetes)

📦 Instalación y Uso
Para ejecutar este proyecto en tu entorno local, sigue estos pasos:

Clonar el repositorio:

Bash
git clone https://github.com/felipesavi/FsClimaApp.git
cd FsClimaApp
Instalar dependencias:

Bash
npm install
Ejecutar en Android:
Asegúrate de tener un emulador abierto o un dispositivo físico conectado mediante ADB.

Bash
npx react-native run-android
📁 Estructura del Proyecto
componentes src/: Contiene los componentes reutilizables de la interfaz.

activos/: Carpeta con imágenes y recursos visuales.

App.js: Archivo principal de la aplicación.

paquete.json: Definición de dependencias y scripts del proyecto.
